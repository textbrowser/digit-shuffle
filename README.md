Provide the operators (?) and the solutions (a, b, ..., f) to the equations. The software will discover the digits x1, ..., x9.

    x1 ? x2 ? x3 = a
    ?    ?    ?
    x4 ? x5 ? x6 = b
    ?    ?    ?
    x7 ? x8 ? x9 = c
    ||   ||   ||
    d    e    f

xi is an integer in {1, ..., 9}. Digits may not repeat.
? represents an operator from the set {+, -, /, \*}.
